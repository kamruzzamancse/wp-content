== Description ==

WP Child Theme Generator is a powerful yet beginner-friendly WordPress plugin that lets you create a child theme from any installed WordPress theme — no coding required.

Whether you’re a developer or just getting started, this plugin makes it easy to create and customize child themes with just a few clicks. It’s perfect for anyone who wants to safely modify theme files without affecting the parent theme.

== Frequently Asked Questions ==

= 1. Can I safely delete the parent theme after creating a child theme? =
No. The child theme depends on the parent theme for its core functionality and styling. Deleting the parent theme will break the child theme. Always keep the parent theme installed and updated.

= 2. Will I lose my customizations if I update the parent theme? =
No. That’s the main benefit of using a child theme. All your customizations made in the child theme remain intact even when the parent theme is updated.

= 3. Does this plugin work with any WordPress theme? =
Yes. WP Child Theme Generator supports all themes installed on your WordPress site, including third-party and custom themes.

= 4. Can I uninstall the plugin after creating a child theme? =
Yes. Once the child theme is created, the plugin is no longer required for its functionality. You can safely uninstall or deactivate the plugin.

= 5. Do I need coding knowledge to use this plugin? =
No. The plugin is designed to be beginner-friendly and requires no coding skills. You can create and customize child themes with just a few clicks.

= 6. Will this plugin affect my website’s performance? =
No. WP Child Theme Generator is lightweight and performance-friendly. It won’t slow down your website.

= 7. Can I customize child theme details like name and author? =
Yes. The plugin offers a Custom Child Theme Creator that lets you modify details such as the theme name, version, author, and author URL before generating the child theme.

== Installation ==

= Using The WordPress Dashboard =  
1. Navigate to the 'Add New' in the plugins dashboard  
2. Search for `WP Child Theme Generator`  
3. Click `Install Now`  
4. Activate the plugin on the Plugin dashboard  

= Uploading in WordPress Dashboard =  
1. Navigate to the 'Add New' in the plugins dashboard  
2. Navigate to the 'Upload' area  
3. Select `wp-child-theme-generator.zip` from your computer  
4. Click 'Install Now'  
5. Activate the plugin in the Plugin dashboard  

= Using FTP =  
1. Download `wp-child-theme-generator.zip`  
2. Extract the `wp-child-theme-generator` directory to your computer  
3. Upload the `wp-child-theme-generator` directory to the `/wp-content/plugins/` directory  
4. Activate the plugin in the Plugin dashboard
